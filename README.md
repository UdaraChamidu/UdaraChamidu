With 4 years of industry experience and a passion for **Machine Learning (ML)**, **Deep Learning (DL)**, **Generative AI**, **Natural Language Processing (NLP)**, **Transformers**, **Retrieval-Augmented Generation (RAG)**, and **AI Agents**, I build intelligent systems that solve real-world problems. ## 💻 Projects

Here are some of my standout projects from [my GitHub](https://github.com/UdaraChamidu) and CV:

### 🩺 [Medical Chatbot](https://github.com/UdaraChamidu) *(Ongoing)*
A multimodal AI chatbot using FastAPI and Groq’s vision-language models to analyze medical images and symptoms, providing diagnostic insights.
- **Tech**: FastAPI, Groq, Vision Transformers, LLMs
- **Features**: Image analysis, symptom-based diagnostics, web interface
- **Status**: In development

### 📑 [RAG-Based PDF Chatbot](https://github.com/UdaraChamidu)
A web-based AI assistant for querying PDF documents using natural language, leveraging RAG architecture.
- **Tech**: Streamlit, LangChain, FAISS, Hugging Face, LLaMA 3
- **Features**: Semantic search, document Q&A, user-friendly interface
- **Status**: Completed

### 👁️ [Eye Disease Classification with Chatbot](https://github.com/UdaraChamidu) *(Ongoing)*
An AI diagnostic system for classifying eye diseases (Glaucoma, Diabetic Retinopathy, Cataract) using OCT images and patient symptoms.
- **Tech**: Vision Transformer (ViT), LLMs, Multimodal Fusion, LangChain
- **Features**: Image classification, conversational diagnostics, transparent insights
- **Status**: Research project in progress

### 💸 [Bitcoin Price Predictor](https://github.com/UdaraChamidu/AI-ML)
A deep learning model using LSTM to predict Bitcoin prices, deployed via a Flask web app.
- **Tech**: TensorFlow, LSTM, Flask, Pandas
- **Features**: Time-series prediction, user-friendly interface
- **Status**: Completed

### 🐾 [Dog vs. Cat Classifier](https://github.com/UdaraChamidu/AI-ML)
A CNN-based binary image classifier to distinguish cats and dogs from a 10,000-image dataset.
- **Tech**: TensorFlow, CNN, Python
- **Features**: Image preprocessing, high-accuracy classification
- **Status**: Completed

### 📦 [Courier Management System](https://github.com/UdaraChamidu/Courier-Management-System)
A modular full-business software framework for managing courier services.
- **Tech**: PHP, MySQL, JavaScript, HTML, CSS
- **Features**: Order tracking, customer management, logistics automation
- **Status**: Actively maintained

### 🚂 [Smart Railway Gate Control System](https://github.com/UdaraChamidu/SmartRail-UI)
An embedded system with GPS tracking and servo motors for automated railway gate control.
- **Tech**: ESP32, Arduino, GPS, React Native (UI)
- **Features**: Real-time tracking, mobile alerts, emergency override
- **Status**: Group project, completed
